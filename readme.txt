== Description ==
<h4>Report For WooCommerce</h4>
<p>
WooCommerce report plugin easily display the list of sold product and list of order. The plugin show the all key information like product name, order status, product total, order date, customer name, customer billing address etc.
</p>
<p>
This report plugin show the sold product information and order information separately. Also you can sort any report by clicking on the header columns of product report and order report. Ability to filter the order and product report base on order date like today order, yesterday order, last 7 day order etc.
</p>




<h4>Key Features</h4>
<ul>
<li>Dashboard show monthly gross sales and net sales.</li>
<li><strong>Order report</strong> show list of all woocommerce order with filter option.</li>
<li><strong>Product report</strong> show the list of all sales product with filter option.</li>
<li>Order Report and Product Report are sorted by report header</li>
<li>Filter order report and product report by today, yesterday, last 7,30,60,90 days. </li>
</ul>
== Installation ==

1. Download <strong>Report For WooCommerce</strong> WooCommerce Plugin to your desktop.
2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
3. Read through the "readme.txt" file thoroughly to ensure you follow the installation instructions.
4. With your FTP program, upload the Plugin folder to the wp-content/plugins folder.
5. Go to Plugins screen in WordPress admin and find the "Sales Report" uploaded Plugin in the list.
6. Click Activate to activate it.
